# 🎓 Exam Score Prediction App

A Machine Learning web application to predict student exam scores based on study habits and environmental factors.

This project was built using:
- Python
- Scikit-Learn
- Streamlit

## 📊 Features
- Input student study data
- Predict exam score using trained ML model
- Interactive web interface

## 🚀 Live Demo
[Click here to access the live demo](https://exam-score-prediction-arf.streamlit.app/)

## 🧠 Machine Learning Model
The model was trained using the **Student Performance Factors dataset** and implemented using a Scikit-Learn pipeline.

Preprocessing includes:
- StandardScaler (numerical features)
- OrdinalEncoder (ordinal features)
- OneHotEncoder (categorical features)
