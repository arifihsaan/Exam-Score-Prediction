# 🎓 Exam Score Prediction

A **Machine Learning web application** that predicts student exam scores based on study habits, lifestyle, and environmental factors.  
This project demonstrates the end-to-end **Data Science workflow**, from **data analysis → model training → web deployment**.

---

## 🚀 Live Demo

Try the application here:

👉 **[Exam Score Prediction](https://exam-score-prediction-arf.streamlit.app/)**

---

## 📊 Project Overview

Many factors influence student academic performance, such as:

- Study hours
- Sleep habits
- Attendance
- Parental involvement
- Access to learning resources

This project uses **Machine Learning** to analyze those factors and predict **student exam scores**.

---

## 🧠 Machine Learning Model

The model was trained using **student performance data** and includes features such as:

- Hours Studied
- Attendance
- Sleep Hours
- Previous Scores
- Tutoring Sessions
- Physical Activity
- Parental Involvement
- Access to Resources
- Motivation Level
- Family Income
- Teacher Quality
- Peer Influence
- Parental Education Level
- Distance from Home
- School Type
- Internet Access
- Extracurricular Activities
- Learning Disabilities
- Gender
